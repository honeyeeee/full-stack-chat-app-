# 💬 Real-Time One-to-One Chat Application

A complete full-stack, real-time chat application built using the **MERN Stack** (MongoDB, Express.js, React, Node.js) and **Socket.io** for instantaneous, bi-directional communication.

## 🚀 Features

* **Real-Time Messaging:** Instant message delivery and receipt using WebSockets (Socket.io).
* **Secure Authentication:** User signup and login functionality to protect chat access.
* **Structured Chat History:** Optimized MongoDB database to store and load messages efficiently.
* **Containerization (Docker):** Application components are fully Dockerized for consistent development and deployment environments.
* **Automated CI/CD Pipeline:** Configured GitHub Actions to automate code linting, building, and deployment workflows.

## 🛠️ Tech Stack

* **Frontend:** ReactJS, HTML5, CSS3
* **Backend:** Node.js, Express.js
* **Real-Time Communication:** Socket.io
* **Database:** MongoDB
* **DevOps & Infrastructure:** Docker, GitHub Actions
